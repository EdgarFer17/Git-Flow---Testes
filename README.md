# Git-Flow---Testes
### git clone : clona um repositorio do github

### git init: inicia o repositorio

### git add . : adiciona todos os arquivos para o repositorio local

### git commit -m: commita esse repositorio 

### git push main: envia o seu repositorio local para o github

### git checkout -b : cria uma branch

### git checkout "nome": troca para a branch criada

### git branch: lista as branchs existente

### git commit --amend -m "commit": para você mudar o commit do seu ultimo commit

### git reset HEAD^ --soft
